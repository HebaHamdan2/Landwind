# Landing Page

A modern and responsive landing page built with Tailwind CSS, DaisyUI, and Flowbite components. The design was implemented based on a Figma prototype, focusing on clean UI, responsiveness, and reusable components.

## Features
- Responsive design for mobile, tablet, and desktop devices
- Built with Tailwind CSS utility classes
- DaisyUI components for faster UI development
- Flowbite interactive components
- Clean and modern user interface
- Figma-based implementation

## Design

The landing page was developed from a Figma design, ensuring a pixel-accurate and responsive implementation.
